# Remote Repository

## What is a Remote Repository?

A Remote Repository is the online version of a Git repository hosted on platforms such as GitHub.

---

## Why Do We Use It?

The Remote Repository is used to share projects, collaborate with other developers, and maintain an online backup of the project.

---

## Example

After creating a commit locally, execute:

```bash
git push origin main
```

The commit is uploaded from the Local Repository to the Remote Repository.

---

## Key Points

- Stored on GitHub.
- Used for collaboration.
- Maintains an online backup.
- Receives commits using `git push`.

---

## Summary

The Remote Repository stores the online copy of a project, making collaboration and backup easier.
